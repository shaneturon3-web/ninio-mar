# Source Rules

## Extraction order

1. Preserve original source title, Drive ID, and date.
2. Extract seed without normalizing language prematurely.
3. Classify anomaly: Signal / Noise / Placeholder / Alias / Compression / Unknown.
4. If Unknown, preserve.
5. Map function before renaming.
6. Apply current canon only after the function is mapped.
7. Move identifiable living-person material into composite, object, weather, or delayed reveal.

## Forbidden transforms

- Do not collapse all sea material into Pacific.
- Do not cluster abuela, bedtime story, niño del mar, Aurora, luz, enfermedad, and omisión.
- Do not use “puerta falsa.”
- Do not turn memoir material into legal proof.
- Do not make the narrator prophet, victim-only, or axe-only.

## Current source hierarchy

Current command > verified Drive material > uploaded file > chat material > memory.
