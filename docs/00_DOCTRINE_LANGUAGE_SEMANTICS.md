# 00 Doctrine — Languages, Semantics & Faux Amis

Source: Google Drive document `Copy of 00 Doctrine Languages, Semantics & Faux Amis`.

## Central rule

Do not translate words.

Translate the semantic function under pressure.

## Operating translation rule

We are not looking for clean equivalence. Each language must preserve the narrative, mechanical, legal, emotional, or comic function that the word had in the original system.

## General doctrine

1. English is often the canonical build language, but not the superior language.
2. Spanish is not translation mode. Spanish is a parallel reading engine.
3. French can be scaffolded, but should not be presented as complete until adapted by function.
4. Route IDs, node IDs, file names, operational names, weird tags, and structural puns are not automatically translated.
5. False friends are not errors by default. They may be semantic traps, misclassification devices, or pressure points.
6. Functional weirdness survives correction.
7. Preserve semantic role over lexical neatness.
8. Preserve the damage a word does.

## Rules for this book

- Draft primary manuscript in Spanish.
- Use English for architecture, editing, and functional translation.
- Let French appear only when it carries Quebec, institutional distance, public-facing coldness, river-memory, or a specific social register.
- Spanglish and malformed phrases may remain if they perform pressure.
- Do not domesticate operational names unless function improves.
- If a phrase is grammatically better but semantically weaker, reject the correction.
- If translation makes the system clearer than intended, damage it again.

## Dangerous words to classify before translating

- record = expediente / registro / acta / historial / prueba.
- report = informe / reporte / acta / expediente.
- file = archivo / expediente / documento.
- folder = carpeta / contenedor fallido.
- archive = archivo / archivo institucional / régimen de preservación.
- evidence = prueba / evidencia / indicio / material probatorio.
- order = orden / mandato / secuencia / clasificación.
- actual = real, not current.
- current = actual / corriente, según contexto.
- preserve = preservar / conservar / mantener, never automatically save.
- latch = pestillo / cerrojo / traba / seguro / latch, depending on scene.
- safe = seguro / caja fuerte / a salvo.
- route = ruta / vía / secuencia / lectura.
- gap = brecha / hueco / vacío / falta.

## Iron fist wrapped in foam

A sentence may enter softly and then become structurally brutal. The surface must not scream if the mechanism can tighten quietly.

Example mode:

- Soft surface: “Los niños no vieron el evento.”
- Iron underneath: “Aprendieron el clima que los adultos llamaron silencio.”

## Translation order

1. Spanish draft.
2. Function map.
3. English translation by pressure, not lexical equivalence.
4. Review for over-cleaning.
5. Re-damage if necessary.
