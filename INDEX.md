# INDEX — ninio-mar

Working title: **Do Not Graft From Memory Without Testing Life**
Public/book title branch: **El niño que vino del mar**

## Operating decision

- Edit in English.
- Write primary manuscript in Spanish.
- Translate into English after Spanish stabilizes.
- Follow `docs/00_DOCTRINE_LANGUAGE_SEMANTICS.md` until modified.
- Expansion now. Form later. Cutting later. Safety pass later.

## Core doctrine

Do not translate words. Translate the semantic function under pressure.

Spanish is not translation mode. Spanish is a parallel reading engine.

If a phrase is strange but performs pressure, keep the strangeness. If a translation makes the system clearer than intended, damage it again.

## Form model

This is a Möbius book with four braided countercurrents:

1. **Autobiographical current** — what happened, or what can be safely approached.
2. **Allegorical / educational current** — maintenance, trees, latch, ledger, systems.
3. **Fragment current** — overheard adult speech, family secrets, damaged memory, oral archives.
4. **Retrospective current** — Leuchtturm, the house, trees, final reclassification.

## Development mode

Chronological order is not binding in this pass. The current goal is expansion. If a chapter blocks, log the blockage and move to the next chapter. After a pass, review blockages because some may resolve through neighboring chapters.

## Multilingual arsenal

Use Spanish as primary engine. Use English as architectural/editorial scaffold. Use French only when it carries Quebec, institutional pressure, social distance, or the river-memory of Montreal. Preserve Spanglish and broken-function phrases when they carry signal.

Iron fist wrapped in foam = a sentence that lands gently enough to enter the reader, then tightens after it is already inside.

## Canon constraints

- El manzano de invierno, not Grandmother Pear.
- La enfermedad del injerto, not blight.
- No Pacific origin for the child from the sea.
- “Vino del mar” remains broad, genealogical, multigenerational.
- Montreal phrase: “Montreal, donde el río finge memoria de mar.”
- San Lorenzo appears only as river-memory of sea.
- No puerta falsa imagery.
- Do not cluster abuela + bedtime story + niño del mar + Aurora + luz + enfermedad + omisión.
- Do not cluster legal documents + medication + omission + altered states + domestic utensils.
